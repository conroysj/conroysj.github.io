Welcome! I'm Hosting with Ghost on GitHub Pages!!

23 March 2015

Hello, world!

¿Cómo va la cosita?

You're very welcome to join me as I blog about software and the world around me. I hope you'll find this material interesting, helpful, or perhaps insightful. I will try to the best of my ability to give credit where credit is due. For starters, this site is in part made possible by:

The Ghost Blogging Platform - a super simple, minimal text editor that can be hosted on GitHub Pages. I leveraged Buster to create a static instance of Ghost, which I run locally to edit my page. This Stack Overflow post gave me the resources to iron out my redirecting woes in no time.

If you have or develop questions while reading my posts, simply reach out to me - I really do appreciate every bit of feedback.
